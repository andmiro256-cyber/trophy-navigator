Возможности программы
🗺 Карты
• 26+ базовых карт (Google, Яндекс, OSM, ESRI, ГГЦ, Генштаб и др.)
• 11 оверлейных слоёв (GPS треки, тропы, подписи, Wikimapia)
• Пользовательские карты — добавьте любой тайловый сервер
• Каталог карт обновляется с сервера автоматически
• Скачивание карт для офлайн (RMaps/OsmAnd) с наложением слоёв
📍 Точки (Waypoints)
• Создание точек кликом на карту или по координатам
• Наборы точек — группировка по категориям
• Радиус сближения — настраиваемый для каждой точки
• Иконки Garmin — автоматическое преобразование символов
• Импорт/экспорт: GPX, WPT (OziExplorer), CSV
〜 Треки
• Рисование треков на карте с отображением длины
• Редактирование: перетаскивание точек, удаление, разбиение
• Импорт GPX/PLT с сохранением высоты и скорости
• Настраиваемый цвет, толщина, размер точек
⚑ Маршруты
• Создание маршрутов по контрольным точкам
• Перетаскивание точек в списке (drag & drop)
• Расстояние и азимут между точками
• Импорт/экспорт: GPX, RTE (OziExplorer)
🔍 Поиск и роутинг
• Поиск по адресу (Nominatim / OpenStreetMap)
• Результаты поиска → сразу в маршрут (A/B) или точку
• Маршрут по дорогам (OSRM) — расстояние, время, скорость
• ПКМ на карте → «Маршрут отсюда / сюда»
📐 Инструменты
• Линейка (M) — измерение расстояний с азимутом
• Блокировка объектов (K) — защита от случайного перемещения
• Очистка карты — удаление всех данных с подтверждением
• Undo (Ctrl+Z) — отмена последнего действия
⟲ Синхронизация
• Синхронизация с Android (Trophy Navigator) через сервер
• Привязка по email — одна учётка на все устройства
• Выбор данных для отправки/получения
⚙ Настройки
• Цвета: точки, треки, маршруты, радиус, OSRM
• Прозрачность заливки радиуса
• Размер иконок, толщина треков
• Автосохранение каждые 30 сек
⌨ Горячие клавиши
• H / Пробел — навигация   W — добавить точку
• T — новый трек   R — новый маршрут
• M — линейка   K — блокировка
• / — поиск   D — скачать карту
• L — выбор слоёв   Ctrl+S — сохранить GPX
• Ctrl+Z — отмена   ESC — сброс режима
